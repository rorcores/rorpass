# RorPass

Lightweight macOS menu bar password generator.

## Build

```bash
chmod +x build.sh
./build.sh
```

## Run

```bash
open build/RorPass.app
```

## Install

```bash
cp -r build/RorPass.app /Applications/
```

To launch on login, add RorPass to **System Settings > General > Login Items**.

## Usage

Click the key icon in the menu bar to open the popover. Drag the slider to set password length (1–35, default 22). Toggle character sets on/off. Click the copy icon to copy the password to your clipboard.
